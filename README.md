# Puppeteer Demo Project
## Project Setup
### Install Puppeteer
In terminal: 
```dash
npm i puppeteer
```
**may refer to [Puppeteer Installation Docs](https://pptr.dev/guides/installation) for more info.

## Features
1. Open a website
2. Click on an element & search bar
3. Input text
4. Click button on keyboard
5. Extract text
6. Save into JSON
7. Close browser

**For more functions, visit [Puppeteer API Page](https://pptr.dev/api/puppeteer.page).

## To identify the Element 
1. Go to the website
2. Right click
3. Click "Inspect"
4. Ctrl + Shift + C
5. Click on the desired element
6. Look at its code to find the type of element (e.g a, button) and unique id of the desired element (e.g. id, class, name)

## How to Run
Ensure you have node.js and run:
```bash
node .\index.js
```
